# Functions re-exported from Momocs2

These functions are re-exported from Momocs2 for convenient access in
Momstats. Click the function names below to see their full documentation
in Momocs2.

## Usage

``` r
fold(.data, ..., .class = NULL)

unfold(x, ...)

get_coe_cols(df, .cols = NULL)

get_all_coe_cols(df)
```

## See also

- [`Momocs2::fold()`](https://momx.github.io/Momocs2/reference/fold.html) -
  Fold multiple columns into a list-column

- [`Momocs2::unfold()`](https://momx.github.io/Momocs2/reference/unfold.html) -
  Unfold a list-column into multiple columns

- [`Momocs2::get_coe_cols()`](https://momx.github.io/Momocs2/reference/get_coe_cols.html) -
  Get single coefficient column name

- [`Momocs2::get_all_coe_cols()`](https://momx.github.io/Momocs2/reference/get_coe_cols.html) -
  Get all coefficient column names
